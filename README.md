# XV de Pau

Invitacion web de una sola pagina para los XV de Pau.

## Abrir

```bash
python3 -m http.server 4173
```

Luego abre:

```text
http://127.0.0.1:4173
```

## Archivos importantes

- Musica principal: Spotify oculto controlado por el boton del disco.
- Reglas de Firestore: copia el contenido de `firestore.rules` en Firebase Console > Firestore > Reglas y toca Publicar.
- Video incrustado: `video/mensaje-especial.mp4`
- Foto del lugar: `img/lugar.jpeg`
- Fotos vestido blanco: `img/fotoxv1.JPG`, `img/fotoxv3.JPG`, `img/fotoxv4.JPG`, `img/fotoxv7.JPG`, `img/fotoxv8.JPG`
- Fotos vestido rojo: `img/fotoxv2.JPG`, `img/fotoxv5.JPG`, `img/fotoxv6.JPG`, `img/fotoxv9.JPG`, `img/fotoxv10.JPG`

## Secciones

Portada, video, informacion del evento, temporizador, vestimenta, mundial, regalos, confirmar asistencia y footer.
